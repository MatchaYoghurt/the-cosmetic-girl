# 名字

最近這幾年性別認同議題浮現，尤其是在大學院校裡面最為明顯3。很多人在 zoom 名字後面會加註自己喜歡的第三人稱。甚至連印識別證的時候，都可以選擇使用 preferred name 而不是 birth name。美國進步派的人還蠻在意這件事情的，甚至有祖母使用 deadname 叫孫女，被認定為虐童而遭起訴，一審還判有罪。後來上訴之後祖母勝訴，目前正在告學校跟醫院。這個案例中還有隔代教養跟學校與醫院跳過監護人，讓未成年人進行性別轉換的荷爾蒙治療的問題，有點扯太遠了。

總之變身成為魔法少女（？）之後應該要有個名字4，我當然就認真找找怎樣的名字才是好好的代表我。畢竟我連出國都不打算另外取名，反正中文名還不致於念不出來，各位老師同學們就自己想辦法念出我的名。

找名字這種事情，最方便的辦法就是去那種給新手爸媽找名字的網站5。同時還可以分析名字靈數還是什麼的。一搜下去才發現 Shoushan 是登記有案的名字。

## Shoushan

在國外，我的身分的官方文件就是中華民國（台灣）護照，上面寫的 first name 是 SHOU-SHAN。就發音來說非常接近，聲調上四聲接一聲的時候稍微不把四聲發完整也不會太奇怪，所以對於不會使用聲調的人來發出來的音，不會與中文差太遠。

發音沒問題，但是拼字就會有些問題了。很多姓名系統上面是不能接受 26 個字母以外的任何符號的6，所以 hyphen 不被系統接受，有些就會直接變成空格。可是空格的話，就會把 shou 當作 first name，shan 當成 middle name。這個問題在麻州的駕照跟某一家銀行。好處就是如果看到署名要給 Shou 的廣告信，大概是可以猜出資料從哪裡流出去的7。

機票上就是直接把中間的符號跳過，就是 SHOUSHAN，說起來也還不錯，因為剛好沒有母音開頭或結尾，所以不會有音節分開的問題。有些直接這樣連起來的名字會讓人搞不清楚到底哪個母音要跟哪個子音。

一方面是因為護照上面是有 hyphen 的，另一方面是這也是台灣人獨有的寫法。同樣是華文名，中國人的名字就不會有 hyphen，我也是這樣告訴其他國家的人這個分辨中國人與台灣人的小撇步。所以我在需要使用寫名字的地方，都會加上 hyphen 。一直到最近才改了。雖然官方名字我會使用 Shou-Shan，但是如果有地方讓人填 preferred name ，我會寫 Shoushan 。

## 百合花

改的原因就是發現 Shoushan 是登記有案的名字。在亞美尼亞 (Armenia)，這是一個不常見但是確實有在使用的名字，甚至在 linkedin 上面也可以找到幾個人。

百合花在亞美尼亞語的發音就是 Shoushan，他們也會拿這個字當作女子名。說法是跟 Susan 同源，意義上是 Lily 8。

所以原來有一個地方的人用跟我的名字一樣的發音來稱呼百合花！有了這層連結，加上百合在日本 ACG 圈中又有女女戀的延伸意思，那我不叫ゆり真是對不起亞美尼亞百合花了9。

雖然 Yuri 這個拼寫在俄語系裡面是跟 George 同源的男子名，不過問題不大，我能接受別人誤會我是男的（？）。於是我所想要的裏之 preferred name 就這麼定下來了。

## Yuri

拼音決定了，那漢字該怎麼寫呢？如果要完全的一個假名，我也是按照雯麗大人所開創的神樂坂家族，以神樂坂作為偽娘名的姓氏10。於是我就搜尋日文拼音對照的漢字名來尋找，這個名字還蠻菜市場的，所以也衍伸出非常多種的漢字寫法。經過複雜的決策判斷，我覺得網路上使用『憂莉』，現實（？）上使用『優璃』。

本來我是因為喜歡璃這個字，搭配總是有那麼一點的哀傷感，想要用『憂莉』來作為名字。2024年過年有書法老師來免費寫春聯的時候，我還請老師寫了以「憂」跟「璃」的藏尾春聯11。不過後來用日本姓名算命網站算一算，『優璃』的命好非常多12，憂字終究是不太妙啊，搭配我的優雅生活小物的創業，還是用優吧。

## 小柔

其實我老早就在尋找理想的名字了，之前那個讓小南寫祭品紀念文的部落格就叫做『麻煩請叫我小柔』。副標題是『無欲則剛』，邏輯上來說是正確的，柔非剛，非剛則非無欲。慾望有點多的就是小柔啦13。

雖然豬熊柔很強，但是這邊的小柔跟那個小柔無關。
